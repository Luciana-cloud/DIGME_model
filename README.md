# DIGME Model Repository

Repository for the **DIGME project**, which investigates how **soil respiration** responds to **soil water potential** under **drought** and **ambient** moisture treatments. The repository includes data, model implementations, and processing scripts used across all project sites.

---

## Repository structure

- **General_data**  
  Main datasets required to run the analysis and models.

- **output_files**  
  Final and intermediate outputs generated during data processing.

- **Water_Potential**  
  Van Genuchten model fits and parameter estimates for all project sites.

- **Manzoni_Respiration_Model**  
  Implementation of the Manzoni et al. (2012) model linking soil respiration fluxes to soil water potential.

- **DIGME_data_processing.R**  
  Main script containing the full data-processing workflow.
